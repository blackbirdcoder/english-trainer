Тренер по английскому

Настройки

Файл словаря должен иметь расширение ".txt". Содержимое файла должно быть организовано с разделением двоеточием ":" между словами. Рекомендуется использовать четное количество слов. Используйте минимум восемь пар слов. Каждое слово на английском языке должно быть разделено от их перевода на другом языке двоеточием. Каждая пара слов должна быть на новой строке.
Например:
satellites : спутники
highlighted : выделено

lorem ipsum dolor sit amet
